# 诗栖 App 高保真原型

## 用户体验分析

目标用户是希望系统化阅读、理解和背诵古诗词的学习者。核心需求包括每日低负担读诗、按主题快速探索、查看原文/注释/译文/赏析、听朗读辅助记忆、收藏沉淀诗单，以及查看个人学习轨迹。

核心交互链路：

1. 今日页进入每日推荐，完成朗读或赏读。
2. 发现页通过搜索、分类、诗人入口找到诗词。
3. 详情页在原文、注释、译文、赏析之间切换，并可收藏和播放朗读。
4. 练习页通过听写、补句、复习错句形成记忆闭环。
5. 收藏页管理诗词、笔记、离线诗库。
6. 我的页查看偏好和成就，设置页调整阅读、朗读和提醒。

## 产品界面规划

| 文件 | 页面 | 主要功能 |
| --- | --- | --- |
| `index.html` | 原型总览 | 使用 iframe 平铺所有手机界面 |
| `home.html` | 今日 | 每日推荐、进度、继续阅读、诗境推荐 |
| `discover.html` | 发现 | 搜索、分类、诗单、热门诗人 |
| `poem.html` | 诗词详情 | 原文、注释、译文、赏析、朗读、收藏 |
| `practice.html` | 练习 | 连续学习、听写挑战、训练计划 |
| `library.html` | 收藏 | 收藏诗词、笔记、离线内容、诗单 |
| `profile.html` | 我的 | 用户数据、偏好、徽章、服务入口 |
| `settings.html` | 设置 | 账号同步、阅读主题、字号、朗读、提醒 |

## UI 设计要点

- 设备尺寸按 iPhone 15 Pro 逻辑屏幕 `393 × 852` 模拟。
- 每个页面包含 iOS 状态栏、Dynamic Island、导航栏与底部 Tab Bar。
- 视觉系统采用宣纸底色、墨色正文、朱砂强调、青绿信息色，配合宋体风格诗文排版。
- 图片素材使用真实远程图片，主要来自 Unsplash。
- 交互包含收藏状态、朗读播放状态、分类/分段选中、练习答题反馈、设置开关等轻量原型行为。

## 使用方式

直接打开 `index.html` 查看全部原型。也可以单独打开任意页面 HTML 文件进行开发拆解。
