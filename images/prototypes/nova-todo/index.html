NOVA Todo 高保真原型

用户需求聚焦在快速捕获、今日执行、项目推进、时间安排和专注完成。核心交互逻辑采用“收集 -> 归类 -> 排期 -> 专注 -> 复盘”的闭环，所有界面均以独立 HTML 文件实现，主入口仅通过 iframe 平铺展示，便于设计走查和开发拆分。

1. 用户体验分析

高频用户需要少打字、少跳转、快决策。原型提供智能捕获、优先级标签、时间块、完成反馈和专注模式。

2. 产品界面规划

关键页面包含今日、收件箱、任务详情、项目、日历、专注、我的设置，覆盖 TODO App 的主要业务闭环。

3. 高保真 UI 设计

使用 iOS 状态栏、底部 Tab、圆角设备框、真实照片、FontAwesome 图标和现代任务卡片系统。

4. 开发可用结构

共享样式位于 assets/styles.css，轻量交互位于 assets/app.js，每个界面文件都可单独打开和迭代。

home.html · 今日总览 执行入口
inbox.html · 收件箱 快速捕获
task-detail.html · 任务详情 执行拆解
project.html · 项目看板 协作推进
calendar.html · 日历排期 时间管理
focus.html · 专注模式 沉浸执行
profile.html · 我的设置 账户偏好
