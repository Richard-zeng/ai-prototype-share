给独居 90 后的每日安全确认 App。核心不是制造恐惧，而是用黑色幽默降低“报平安”的心理负担，让亲友、设备和上门服务形成闭环。

iPhone 15 Pro Tailwind CSS FontAwesome iframe 平铺

1. 用户体验分析

目标用户是独居、社恐、作息不稳定的年轻人。痛点是“每天报平安很尴尬”“亲友不知道何时该介入”“真实异常缺少轻量流程”。核心路径为每日打卡、超时提醒、亲友确认、骑手安全查看、必要时善后协作。

2. 产品界面规划

信息架构按用户端和骑手端拆分：用户端包含首页、报平安、预警、订单详情、社区、我的、新手设置；骑手端聚焦接单、订单风险、授权边界和收益。

3. 高保真 UI 设计

视觉使用薄荷绿、珊瑚红、薰衣草浅色和深色骑手页。所有 App 屏模拟 iOS 状态栏、动态岛、底部 Tab Bar、卡片、按钮、列表、时间线与真实图片。

4. HTML 原型实现

每个界面都是独立 HTML 文件，入口页只通过 iframe 嵌入，不复制子页面代码。可直接交给开发拆组件、抽数据和接接口。
