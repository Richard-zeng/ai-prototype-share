诗栖 App 高保真原型

核心链路覆盖今日读诗、发现搜索、诗词详情、背诵练习、收藏书架、我的与设置。每个界面都是独立 HTML 文件，主入口仅用 iframe 平铺展示，尺寸按 iPhone 15 Pro 逻辑屏幕模拟。

设备：iPhone 15 Pro
画布：393 × 852 CSS px
技术：HTML + Tailwind CDN + FontAwesome
